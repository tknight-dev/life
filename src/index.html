Rules
Conway's Game of Life:
  • Diagonal, Horizontal, and Vertical cells considered
  1.  Dies: Any live cell with fewer than two live neighbors (underpopulation)
  2. Lives: Any live cell with two or three live neighbors
  3.  Dies: Any live cell with more than three live neighbors (overpopulation)
  4.  Born: Any dead cell with exactly three live neighbors

Limitations:
  • Iterations per Second (i/s)
    • Max: 1,024
    • Min: 1
  • Table size (16x9 aspect ratio)
    • Max: is 2,032 by 1,143 (≤ ~9 MB)
    • Min: is    48 by    27 (≤ ~5 KB)
